## Installation
```
curl -sSL https://raw.githubusercontent.com/Trabbit0ne/GBash/refs/heads/main/main.sh -o $(dirname $(command -v bash))/gbash && chmod +x $(dirname $(command -v bash))/gbash
```

## Usage
``
gbash <file>
``
